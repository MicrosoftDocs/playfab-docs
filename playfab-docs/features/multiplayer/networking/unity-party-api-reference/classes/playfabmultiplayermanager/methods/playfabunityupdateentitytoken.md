# UpdateEntityToken

Updates the Entity token for the current local user.

## Syntax

```csharp
public void PlayFabMultiplayerManager.Get().UpdateEntityToken(entityToken);
```

### Parameters

**entityToken** string

The Entity token associated with the local user.

## Return value

None.
