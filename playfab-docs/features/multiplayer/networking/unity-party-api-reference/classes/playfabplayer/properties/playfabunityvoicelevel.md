# VoiceLevel

Gets or sets the volume of the player. The value is a float between 0 and 1.

```csharp
public float PlayFabPlayer.VoiceLevel { get; set; }
```

## Property Value

VoiceLevel

[float]()

A value from zero to one inclusive.
