# JoinNetwork

Joins this player to the specified network.

## Syntax

```csharp
public void PlayFabMultiplayerManager.Get().JoinNetwork(string networkId);
```

### Parameters

**networkId** string

A string that contains the information necessary for a player to join a network.

### Return value

None.
