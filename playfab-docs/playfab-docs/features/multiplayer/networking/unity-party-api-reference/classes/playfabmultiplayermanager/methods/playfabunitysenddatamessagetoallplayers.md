# SendDataMessageToAllPlayers

Broadcasts a data message to all players.

## Syntax

```csharp
public void PlayFabMultiplayerManager.Get().SendDataMessageToAllPlayers(buffer);
```

### Parameters

**buffer** byte[]

A buffer that contains the data to send.

## Return value

None.
